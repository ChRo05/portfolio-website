## 記入ガイド（各列の意味）

- **守秘**: 顧客名・実データ・具体業務が含まれるか → `あり` / `なし`
- **公開可否**: コードをそのまま or 整理すれば公開できるか → `可` / `要改修` / `不可`
- **見せ方**: `公開コード` / `成果のみ（ぼかし）` / `載せない`
- **優先**: ポートフォリオに載せたい度 → `高` / `中` / `低`
- **強み区分**: ①業務ドメイン / ②自動化RPA / ③AI・LLM のどれに当たるか（複数可）
